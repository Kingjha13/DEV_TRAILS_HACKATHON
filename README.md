<div align="center">🛡️ ShieldNet

AI-Powered Parametric Micro-Insurance for Gig Workers

When disruption happens, protection should be instant.

</div>---

🚨 Problem

Gig workers such as delivery riders and drivers often lose income due to conditions beyond their control like heavy rainfall, poor air quality, or traffic disruptions. Even a few hours of disruption can directly impact their daily earnings.

Traditional insurance systems are not designed for this reality. They are slow, manual, and not built for real-time situations.

---

💡 Solution

ShieldNet is a real-time micro-insurance platform that enables fast, data-driven protection.

Instead of relying on manual claims, the system evaluates conditions and responds automatically through a connected mobile and backend architecture.

---

⚙️ How It Works

User registers → Risk score calculated → Policy selected
↓
Disruption detected → System evaluates conditions → Response triggered

---

🚀 Key Features

- Smooth onboarding and registration flow
- Risk score calculation using backend services
- Dynamic premium and coverage selection
- Real-time disruption-based system response
- Dashboard with alerts and activity tracking
- Integrated mobile and backend architecture

📱 App Flow

1. User registers using phone number
2. User profile is stored in backend
3. Risk score is generated
4. User selects a plan
5. Dashboard displays system status and updates
